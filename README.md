# fsociety-vpn-docs

Privacy policy and terms of service for [fsociety vpn](https://github.com/t3rmynal/fsociety-vpn).

These two files are fetched as raw markdown by the client on every launch and rendered inside the TUI:

- `privacy.md`
- `terms.md`

Do not edit these files via the GitHub web UI. If you want to suggest a change, open a PR in the main [fsociety-vpn](https://github.com/t3rmynal/fsociety-vpn) repository describing the proposed wording, and the operator will mirror the accepted version here.

A 1 hour cache and an embedded fallback copy live inside the binary, so the app keeps working when GitHub is unreachable.
